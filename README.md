# 2048-clone
[Original game by Gabriele Cirulli](https://play2048.co/). I've just cloned the game for fun and testing my code skills.

## Installation
```bash
git clone https://github.com/zronev/2048-clone.git
cd 2048-clone
npm install
```

## Start & watch
```bash
npm start
```

## Simple build for production
```bash
npm run build
```
